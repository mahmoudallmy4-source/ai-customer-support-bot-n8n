# AI Customer Support Bot

## Overview

An AI-powered customer support bot built with n8n. It automatically receives Telegram messages, generates AI responses, remembers conversation context, and can escalate customers to a human agent when needed.

---

## Features

- 🤖 AI-powered responses
- 💬 Telegram integration
- 🧠 Conversation memory
- 👨 Human handoff support
- 📊 Google Sheets integration
- ⚡ Built with n8n

---

## Tech Stack

- n8n
- Groq API
- Telegram Bot API
- Google Sheets
- AI Agent
- Memory
